## 三、素材需求详情

### A1 · 北方四合院破旧版

| 属性 | 内容 |
|------|------|
| **用途** | 主场景1 — 破败前院（杂草丛生） |
| **剧情对应** | 第一幕·扫尘 |
| **建议尺寸** | 1920×1080 |
| **最终文件名** | `courtyard_broken.png` |
| **存放路径** | `res://assets/images/backgrounds/courtyard_broken.png` |

**画面要求：**
- 中国北方传统四合院前院，极度荒凉破败，废弃多年无人居住
- 画面近景偏左为广亮大门（门扇退在门洞内侧，门前有宽敞门廊/门墩），大门内侧立有青砖影壁（独立墙体，非八字墙）
- 大门右侧（画面偏右）为东厢房（木工作坊，门窗破损，工具散落）
- 前院青砖墁地被杂草完全覆盖，杂草高至膝盖、枯黄杂乱，多种野草交错丛生，落叶堆积成层
- 画面中央偏右有一棵石榴树已完全枯死，主干扭曲龟裂，枝条光秃无叶，无任何果实
- 天空为阴沉铅灰色，云层厚重压抑，无阳光，整体色调低饱和、灰暗、冷褐
- 墙面有剥落、霉斑，台阶断裂，门窗油漆剥落露出朽木
- 横版构图，游戏场景视角，2D手绘风格，中国水墨淡彩

**使用节点：**

| 场景 | 节点路径 | 操作 |
